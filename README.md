# Algorithmic Credit Risk & Portfolio Optimization Platform

An enterprise-grade financial analytics platform designed to solve the commercial trilemma of automated lending: maximizing portfolio returns, enforcing demographic fairness, and maintaining transparency for central bank audits.

## Tech Stack & Architecture
- **Data Engineering (SQL):** PostgreSQL star-schema modeling for multi-variable staging and data validation.
- **Machine Learning (Python):** Random Forest ensemble classifiers achieving an **87.22% PR-AUC score**, outperforming the baseline by **+18.03%**.
- **Stress Testing (R):** Vectorized macroeconomic sensitivity simulations proving a **14.23% reduction in toxic assets** under a 15% inflation shock scenario.
- **Executive UI (Tableau):** Interactive dashboard control layers mapped to a three-tier 14% automated risk threshold policy.
