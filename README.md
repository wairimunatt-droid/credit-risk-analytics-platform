# Algorithmic Credit Risk & Portfolio Optimization Platform

An enterprise-grade financial analytics platform designed to solve the commercial trilemma of automated lending: maximizing portfolio returns, enforcing demographic fairness, and maintaining total compliance transparency for central bank audits.

## Tech Stack & Architecture
Data Engineering (SQL): PostgreSQL database view layer (v_clean_credit_risk) enforcing strict automated domain bounds to eliminate staging anomalies across 28,632 validated records.
Machine Learning (Python): Scaled Random Forest ensemble classifier achieving a final 0.8909 PR-AUC score, delivering a +16.25% predictive lift over the converged baseline benchmark.
Stress Testing (R): Vectorized macroeconomic sensitivity simulations modeling systemic asset exposure, identifying a 20.88% portfolio default rate and capturing a +114.23% risk volatility surge under a 15% inflation shock scenario.
Executive UI (Tableau): Production-grade control dashboard layers utilizing data extracts to monitor an optimized 14% automated risk threshold policy alongside a 0.87 Disparate Impact Ratio fairness floor.
